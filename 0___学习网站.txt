主要说的是一些国外的学习网站，国内的略过。

开源网站：

Souceforge

全国最大的一个开源网站，很多有名的开源项目都是在这个网站托管的。

Github

非常好的一个开源网站。

Code Google

谷歌旗下的一个开源项目托管网站，资源很丰富，文档也很齐全，甚至还有中文文档。

CodePlex

微软旗下的一个开源项目托管网站。

Codeproject

含有丰富的开源程序，主要是针对的某些功能封装的开源程序，含有事例程序，项目一般比较小，但很丰富。还有一些技术类的文章，也很有参考意义。

CodeGuru

和Codeproject很像，项目一般也都很小。
